#### 功能列表

目前软件还在开发期，功能更新比较频繁，下面是现在已经想到的功能。

- [x] 批量下载多篇文章(给定链接)
- [x] 批量下载多账号的历史文章(给定账号名称)
- [x] 批量按关键词下载搜索的文章(通过微信搜一搜)
- [x] 将文章下载为: html、mhtml和pdf
- [x] 将文章信息导出到表格，例如正文、标题、时间、阅读数、点赞数等
- [x] 下载文章所有评论并导出到表格文件
- [x] 任务完全自动化，添加后无需人工操作
- [x] 任务意外中断后可无感恢复
- [x] 获取或导出已关注的公众号列表
- [ ] 按账号采集时，支持原创、转载、非付费、首篇、图文等筛选条件
- [ ] 支持账号采集最新一页并实现增量采集
- [ ] 下载文章中的资源: 图片、视频和音频 
- [ ] 获取关注的公众号文章更新(需要登录微信)
- [ ] 按合集下载所有文章
- [ ] 增加requests请求获取文章html，当只需要正文而不需要pdf文件时使用
